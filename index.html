About Our Project

The Smart Milk Dispenser is an IoT-based system built using ESP32 and a 5-inch touchscreen. It enables customers to purchase milk seamlessly, select quantity, and make secure digital payments via Razorpay.

Key Features

  • Touchscreen user interface
  • Flexible quantity selection
  • Digital payments via Razorpay
  • Real-time dispensing control
